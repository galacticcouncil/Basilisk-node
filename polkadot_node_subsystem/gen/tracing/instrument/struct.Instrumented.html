Struct polkadot_node_subsystem::gen::tracing::instrument::Instrumented

source · 
pub struct Instrumented<T> { /* private fields */ }
Expand description

A Future that has been instrumented with a tracing Span.

This type is returned by the Instrument extension trait. See that trait’s documentation for details.
