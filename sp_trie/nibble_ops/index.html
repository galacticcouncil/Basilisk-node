Various re-exports from the trie-db crate. Utility methods to work on radix 16 nibble.

Constants

Single nibble length in bit.
CONTENT_HEADER_SIZE
Size of header.
NIBBLE_LENGTH
Number of child for a branch (trie radix).
NIBBLE_PER_BYTE
Number of nibble per byte.
PADDING_BITMASK
Nibble (half a byte).
SPLIT_SHIFTS
The nibble shifts needed to align. We use two value, one is a left shift and the other is a right shift.
